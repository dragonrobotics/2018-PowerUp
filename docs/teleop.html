teleop module¶

Contains functions for teleop logic.

teleop.drive(control_stick, drivetrain)[source]¶

Drive the robot directly using a joystick.

This function grabs inputs from a joystick, applies axis inversion and deadbanding, and from there passes the control inputs on to swerve.swerve_drive.SwerveDrive.drive().
